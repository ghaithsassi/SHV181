## Building and Installing


### Install from git repository

you will need :
* C++ complier >= c++17

#### using terminal 
    git colne https://github.com/ghaithsassi/SHV181.git
    cd SHV181
### Building classes
    g++ -c word.cpp 
    g++ -c stream.cpp 
    g++ -c file.cpp 
    g++ -c database.cpp 
    g++ - c indexing.cpp
### Building main 
    g++ -c main.cpp
### Building exe file
    g++ word.o stream.o file.o database.o indexing.o main.o
